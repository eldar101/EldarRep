This project is a program for an algorithm that prints an array's k-smallest numbers;

The project consists of:
1) KSmallest - Main program and GUI.
2) RedBlackTree - structure of RB tree.
3) utils - functions that assist with the filling and printing of arrays.
4) Project - proof and explanations of the algorithm.
5) Code - text version of the program.
6) Output - text output examples.
